详细使用指南

文件管理

目录操作

  • 打开目录：选择包含待分类图片的根目录
  • 子目录处理：程序会递归扫描所有子目录中的图片
  • 目录结构：分类后的图片会按类别名创建对应文件夹
  • 移出目录：删除的图片会移动到 "remove" 文件夹

类别管理

  • 新增类别：单个添加或批量添加（逗号分隔）
  • 编辑类别：右键类别按钮选择"编辑"
  • 删除类别：右键类别按钮选择"删除"
  • 快捷键设置：右键类别按钮选择"设置快捷键"
  • 类别限制：类别名最长50个字符，支持中英文

图片浏览与操作

视图控制

功能 操作方法 快捷键 说明
适应窗口 菜单/快捷键 Ctrl+F 自动调整图片大小适应显示区域
放大图片 滚轮向上/菜单 Ctrl + = 放大图片，最大倍数可在设置中配置（默认3倍，最大20倍）
缩小图片 滚轮向下/菜单 Ctrl + - 缩小图片显示
原始大小 菜单/快捷键 Ctrl + 0 显示图片100%原始大小
拖拽移动 鼠标左键拖拽 - 移动图片查看不同区域

分类操作

  • 复制模式：保留原文件，复制到目标类别文件夹（默认）
  • 移动模式：直接移动文件到目标类别文件夹
  • 分类方法：双击类别按钮、使用快捷键或按回车键
  • 多分类模式：同一张图片可分配到多个类别
  • 删除图片：按 Delete 键将图片移动到 "remove" 文件夹
  • 撤销操作：支持撤销分类（单/多分类）和撤销删除，再次点击已分类类别或已删除图片的 Delete 键即可撤销

分类模式详解

单分类模式（默认）
  • 一张图片只能属于一个类别
  • 重新分类会自动从旧类别移动到新类别
  • 类别按钮显示绿色背景表示已分类
  • 适合传统的文件整理需求
多分类模式（新功能）
  • 灵活分类：一张图片可以同时属于多个类别
  • 切换方式：点击工具栏的"→/⇶"按钮切换单/多分类
  • 分类操作：点击类别按钮添加分类，再次点击取消分类
  • 视觉反馈：多分类的类别按钮显示蓝色背景
  • 应用场景：标签化管理，如"风景+日落"、"人物+室内"等
操作 多分类模式行为 单分类模式行为
首次分类 添加到类别列表 直接分类到该类别
已分类的类别 从列表中移除（取消分类） 撤销分类（复制模式移除该文件，移动模式文件返回原目录）
其他类别 同时添加到类别列表 从旧类别移动到新类别
多分类模式使用技巧
  • 标签化思维：把类别当作标签，一张图片可以有多个标签
  • 快速取消：再次点击已分类的类别按钮可快速取消该分类
  • 状态查看：蓝色背景的类别按钮表示当前图片属于该类别
  • 物理文件：图片会被复制到每个分类的文件夹中
  • 模式切换：可随时在单分类和多分类模式间切换

状态与统计

状态标识

  • 已分类：图片已成功分类到某个类别
  • 已移出：图片已移动到移出目录
  • 未处理：尚未分类的图片
  • 进度显示：底部状态栏显示处理进度

实时统计

  • 总数统计：显示图片总数和处理进度
  • 类别统计：每个类别的图片数量
  • 效率统计：分类速度和剩余时间估计

同步与刷新

  • 自动同步：程序会定期检测外部文件变化
  • 手动刷新：按 F5 键立即同步文件状态
  • 智能检测：检测新增、删除、移动的文件
  • 状态保存：工作状态自动保存，重启后恢复

高级设置

快捷键系统

  • 自动分配：数字键 1-9 和字母键 a-z 会自动分配给前 35 个类别
  • 手动设置：右键类别按钮选择"设置快捷键"可自定义
  • 组合键：支持 Ctrl、Alt、Shift 组合键，需手动设置
  • 冲突检测：设置快捷键时会自动检测冲突，避开系统保留快捷键
  • 排序模式：支持按类别名称、快捷键或分类数量排列

性能优化

  • 智能预加载：自动预加载下一张图片提高浏览速度
  • 内存管理：智能释放不需要的图片内存，减少占用
  • 多线程处理：后台线程处理文件操作，界面保持流畅
  • 大图片优化：自动检测大图并优化加载策略
  • 批量处理：支持高效处理数千张图片

网络存储优化

  • SMB/NAS 支持：支持 \\server\share 格式的网络路径
  • 网络缓存：智能缓存网络图片提高访问速度
  • 自动重试：网络操作失败时自动重试3次
  • 连接优化：维护网络连接池提高效率
