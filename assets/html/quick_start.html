快速入门指南

三步快速开始

  1. 选择文件夹：点击"打开目录"选择包含图片的文件夹
  2. 创建类别：点击"新增类别"添加分类标签
  3. 开始分类：双击类别按钮或使用快捷键分类图片

支持的图片格式

JPG, JPEG, PNG, BMP, GIF, TIFF, WebP

核心操作

操作 方法 说明
浏览图片 ← → 键 或 鼠标点击 在图片列表中前后导航
选择类别 ↑ ↓ 键 在类别列表中上下切换选择
分类图片 双击类别按钮 或 Enter键 将当前图片分类到选中类别
缩放图片 鼠标滚轮 或 Ctrl +/- 放大缩小查看图片细节
移动图片 鼠标左键拖拽 移动图片查看不同区域
移出图片 Delete 键 将图片移到移出目录

高效使用技巧

  • 使用快捷键：按数字键 1-9 快速分类到对应类别
  • 文件模式切换：点击工具栏的"⧉/✂"按钮切换复制/移动模式
  • 多分类模式：点击工具栏的"→/⇶"按钮切换单/多分类
  • 撤销分类：单分类模式下，再次点击已分类的类别可撤销分类
  • 撤销删除：已删除的图片再次按 Delete 键可从移出目录恢复
  • 回车确认：选中类别后按 Enter 键快速分类
  • 自动同步：程序会自动检测外部文件变化
  • 状态保存：工作状态会自动保存，重启后恢复
  • 主题切换：在设置中可切换亮色/暗色主题，或设置跟随系统
  • 缩放配置：可在设置中自定义图片缩放范围和全局缩放行为
  • 新手教程：首次使用时会自动引导，也可在帮助菜单重新开始
  • 自动更新：支持在线检查和下载最新版本（需联网）
