## [6.0.0] - 2025-08-14

### 🚀 重大更新
- 自动更新功能：程序内检查更新、下载、校验并一键重启安装
- 统一 update/update.bat：支持程序调用与手动双击（在线/离线）
- 原子替换机制：保留新版本文件名，清理旧版本与安装包
- 兼容中文与URL编码：下载、路径与显示全面中文化
- 日志可控：关键节点 INFO，细节 DEBUG；自动检查可配置

- **主版本号**: 不兼容的 API 修改

- **修订号**: 向下兼容的问题修正

- 🎉 首次发布
- ✨ 新增功能

- 🎨 界面/体验改进
- ⚡ 性能优化
- 📝 文档更新
- 🚫 功能移除
- 🔒 安全修复
